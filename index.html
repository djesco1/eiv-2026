Preguntas frecuentes FAQ

¿Cuándo debo inscribir los cursos? When should I register for classes? +

28 y 29 de mayo, a través del portal MiBanner. May 28–29, through the MiBanner LINK portal.

¿En dónde debo revisar los horarios, prerrequisitos y detalles de los cursos? Where can I check schedules, prerequisites, and course details? +

En la oferta de Cursos. In the Courses listing.

¿Quiénes pueden inscribirse? Who can apply? +

Estudiantes de pregrado y posgrado de Uniandes, así como visitantes de universidades nacionales e internacionales. Undergraduate and graduate students from Uniandes, as well as visiting students from national and international universities.

¿Qué pasa si no puedo inscribir uno de estos cursos? What if I cannot register for a course? +

Haz una solicitud a través de Ajuste de Horario Académico (AjHA). Submit a request through Academic Schedule Adjustment.

¿Puedo tomar más de un curso? Can I take more than one course? +

Sí, siempre que las fechas no se crucen. Yes, as long as the dates don't overlap.

¿Para qué área aplican estos cursos? Which area do these courses apply to? +

El área de aplicación de cada curso se encuentra especificada en la información previa. The application area of each course is specified in its description.

¿Debo inscribir Proyecto Intermedio o el curso de Maestría? Should I register it as Intermediate Project or Master's course? +

Si tu interés es tomar el curso INGENIERÍA CON IMPACTO SOCIAL como Proyecto Intermedio, debes inscribirlo con el código IIND-3006; pero si quieres tomarlo como Electiva nivel Maestría Área de Gestión Organizacional, debes inscribirlo con el código IIND-4362.

El curso es válido para una de las dos opciones, asegúrate de inscribir el curso que necesitas pues no se harán homologaciones para estos casos. If you want to take INGENIERÍA CON IMPACTO SOCIAL as an Intermediate Project, register with code IIND-3006; if you want it as a Master's elective (Organizational Management area), register with code IIND-4362.

The course is valid for only one option. Make sure to register correctly, as no equivalences will be granted.
